Plans

The first working release of the argouml-sql module is planned for around 15 April 2007. With this release you can:

  • Build a class diagram that represents a logical relational data model
  • Generate sql for at least one database system

Roadmap

After the release in April there still is work to be done (in no specific order):
  • Implement code generators for the most common database systems
  • Parsing of SQL DDL scripts
  • Reverse engineering of databases
  • Special attribute notation for relational models (like in real database modelling tools
  • Assist the user in building a relational model (auto-creation of foreign key attributes for associations for example)
  • Auto-create a relational model out of a class diagram
  • perhaps more...
